# GridAligner 開発仕様書 (v1.2.1)

## 1. プロジェクト概要
**GridAligner** は、魚眼レンズや超広角レンズによる幾何学的な画像歪み（レンズ歪み・パース歪み）を、直感的な GUI 操作と AI 自動検出によって補正するためのデスクトップアプリケーションである。

> [!NOTE]
> 格子の自動検出は、**チェッカーボード（市松模様）等のテストパターン**、およびコントラストの明確な人工的な格子を主な対象とする。自然風景や一般的な建造物内の複雑な格子検出は、手動操作（コントロールポイントのドラッグ）による補正を前提とする。

## 2. コア機能の詳細

本ツールの機能は、[ARCHITECTURE_MANIFEST.md](../../ARCHITECTURE_MANIFEST.md) に定義された「レンズ補正先行フロー」および「絶対正規化座標系」の原則に基づき実装されている。

### 2.1 レンズ歪み補正 (Lens Correction)
* OpenCV の円形歪みモデルをベースとした補正エンジン。
* **k1 (第2次係数)**, **k2 (第4次係数)** のスライダー操作によるリアルタイム補正。
* バレル（樽型）およびピンクッション（糸巻き型）の両方に対応。

### 2.2 メッシュ・レクティファイ (Mesh Rectification)
* 自由度の高い格子状メッシュによる局所的な歪み（パース・台形歪み）の補正。
* コントロールポイント（赤丸）をマウスでドラッグし、画像内の格子線に合わせることで直線を復元。
* **Smooth Warp (Physics)**: 物理演算ベースのエネルギー最小化により、数点の補正から周辺のメッシュを滑らかに追従させ、格子の連続性を維持する。

### 2.3 AI 格子自動検出 (Auto-Grid Detection V2)
チェッカーボード認識と投影プロファイル解析を統合したハイブリッドエンジン。
（技術的な採用根拠については [ARCHITECTURE_MANIFEST.md](../../ARCHITECTURE_MANIFEST.md) を参照）

*   **ハイブリッド認識の構成**:
    *   **チェッカーボード・フィッティング**: `cv2.findChessboardCornersSB` 等を用いた高精度認識。
    *   **投影プロファイル解析 (Fallback)**: エッジ密度解析による統計的ピーク判定。
    *   **コーナー・サブピクセル吸着**: `cv2.cornerSubPix` による 1ピクセル以下の精度でのスナップ。

### 2.4 アドバンスド・ユーティリティ
*   **Grid Straighten**: 四隅を基準に、内部の交点を幾何学的に正しい直線上へ強制整列。物理的な紙のしなり等による「うねり」をキャンセルする。
*   **Grid Extrapolation**: 小さなボードから得られたパース情報を画像全体へ拡張。室内壁面の正対化等に有効。
*   **Grid Orientation Correction**: AI が誤認した格子の向きを90度単位で手動修正。外部出力時の整合性を確保。

## 3. ワークフローとパイプライン

本アプリは以下の画像処理パイプラインを持つ。

1.  **入力**: オリジナル画像。
2.  **レンズ補正フェーズ**: 放射歪みの除去（直線の復元）。
3.  **メッシュ変形フェーズ**: 補正済み画像に対するホモグラフィ/局所変形の適用。
4.  **プレビュー生成**: PyTorch の `grid_sample` による高速・高品質なリアルタイム描画。

## 4. 設計方針

本プロジェクトは厳格な **MVC パターン** を採用し、UIフレームワークへの依存を最小限に抑えている。具体的なクラス設計および API 契約については、[src/ARCHITECTURE_MANIFEST.md](../../src/ARCHITECTURE_MANIFEST.md) を参照してください。

- **Model**: 純粋なデータコンテナ。
- **View**: 描画と入力を担当。ウィジェット内部を Controller に露出させない。
- **Controller**: 全ての層の同期とライフサイクル管理を統括。
- **Engine**: 状態を持たない計算ロジック。
